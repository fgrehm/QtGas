h1. QtGas

h2. Description

QtGas is a mobile application designed for touch screen devices to track gas mileage
and vehicle maintenance.

h2. Supported platforms

At the moment it was only tested on a Windows Mobile 6.5.3 phone and I've got plans
to add support for other platforms supported by Qt framework. Please let me know
if you are interested in trying it out on some other platform.

h2. Requirements

h3. Windows Mobile

Visual Visual C++ Runtime for your device.
